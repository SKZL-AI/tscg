# TSCG Website Concept Pack

This folder contains the website concept layer for TSCG. It translates the existing TSCG research, benchmark evidence, product architecture and external website-strategy research into a practical brief for frontend design work.

Start here:

- `01-website-konzept.md` - strategic concept, audience, messaging, sitemap and content system.
- `02-claude-ai-design-brief.md` - copy-ready design prompt for Claude AI / Claude Artifacts.
- `03-research-harmonisierung.md` - how the concept aligns with the TSCG research base and the imported website/AI-portfolio research.

Design principle:

TSCG should present as a credible technical research product, not as a generic AI startup landing page. The site should make the value inspectable within seconds: fewer tool-schema tokens, stronger tool-use accuracy, small-model enablement, MCP/agent integration, reproducible evidence.
